Want to learn Go or get started with TUI (Terminal User Interface) development?

We built `hn-client` not just as a tool, but as a live learning sandbox. It includes a comprehensive `TUTORIAL.md` to teach you Go through real code. 🧵
---
Instead of dry, abstract syntax examples, the guide uses our actual code to explain:
- Packages & Modules (`go.mod`)
- Value vs. Pointer Receivers
- Async execution in Bubble Tea
- Suspension of terminal process buffers
---
The codebase is highly modular, making it a perfect starting point to modify, add features, and learn best practices in terminal UI design.

Plus, you can ask your terminal AI agent (like Claude) to teach you using this sandbox!
