Beautiful terminal typography: How we style HN comments! 🎨

hn-client uses `golang.org/x/net/html` to sanitize comment text, converting HTML links and raw blocks into styled monospace code. It renders nested threads with color-coded indent borders so they are easy to read.

How do you display rich text in terminal apps?
