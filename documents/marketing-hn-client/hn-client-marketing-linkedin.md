🚀 Announcing hn-client — A terminal-native Hacker News Client built in Go!

If you spend a lot of time in the terminal, you probably enjoy reading tech news without leaving your workspace. 

I built hn-client to provide a beautiful, lightweight, and keyboard-driven terminal user interface (TUI) for Hacker News.

Key Features:
1️⃣ Blazing Fast & Concurrent: Non-blocking background workers fetch Top, New, Best, Show, and Ask feeds instantly.
2️⃣ Vim-Native Workflow: Navigate with j/k, filter posts in real-time with /, and read detailed comments with Enter.
3️⃣ Beautiful Nested Comments: Styled with color-coded indentations, sanitized HTML links, and native terminal code rendering.
4️⃣ Persistence: Global read-markings (✓) and [NEU] flags for new comments stored in ~/.hn-history.json.
5️⃣ Browser Integration: Seamless fallback to w3m, lynx, or default GUI web browsers (open with 'o', write with 'w', reply with 'r').

hn-client is completely free, open source, and built using the Bubble Tea TUI framework.

👉 Read the guide & download the binary: https://github.com/aeon022/hn-client
