🚀 Announcing hn-client — A terminal-native Hacker News Client built in Go!

If you spend a lot of time in the terminal, you probably enjoy reading tech news without leaving your workspace. I built hn-client to provide a beautiful, lightweight, and keyboard-driven terminal user interface (TUI) for Hacker News.

Key Features:
• Blazing Fast & Concurrent: Non-blocking background workers fetch Top, New, Best, Show, and Ask feeds instantly.
• Vim-Native Workflow: Navigate with j/k, filter posts in real-time with /, and read comments with Enter.
• Beautiful Nested Comments: Color-coded indentations, sanitized HTML links, and native terminal code rendering.
• Persistence: Global read-markings (✓) and [NEU] flags for new comments stored in ~/.hn-history.json.
• Browser Integration: Seamless fallback to w3m, lynx, or default GUI web browsers.

👉 Check it out on GitHub: https://github.com/aeon022/hn-client
