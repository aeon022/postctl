🚀 Announcing hn-client — A terminal-native Hacker News Client built in Go!

Features:
• Vim-style navigation (j/k, Enter to view comments)
• Switch categories with Tab or 1-5
• Live search & filter with /
• Multi-color nested comments with HTML link sanitization
• Persistent read markings (✓) & [NEU] flags
• w3m/lynx integration for web browsing

Free, open source, and written using Bubble Tea.
👉 https://github.com/aeon022/hn-client
