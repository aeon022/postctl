🚀 Announcing missionctl: Giving local AI agents "hands" via terminal CLI & TUI tools!

No cloud subscriptions or SaaS databases. missionctl is a suite of 6 Go tools running locally with SQLite:
1. mailctl 📧 (Apple Mail)
2. calctl 📅 (Apple Calendar)
3. taskctl 📝 (Reminders)
4. notectl 📓 (Obsidian)
5. budgetctl 💰 (Finance)
6. postctl 🐦 (Socials)

Exposes 43 MCP tools to Claude.
👉 https://github.com/aeon022/missionctl
