📅 Meet calctl: Calendar management & scheduling for terminal power users.

Switching to web calendars is a productivity leak. calctl bridges Apple Calendar locally, using a keyboard-driven Bubble Tea monthly grid.

Features:
• macOS EventKit bridge (Apple Calendar)
• Calculate free gaps instantly with `calctl free`
• Import events from Markdown/YAML
• 7 MCP tools for Claude

👉 https://github.com/aeon022/calctl
