📅 Manage your calendar and find free slots directly in your terminal. Meet calctl!

A Go binary that bridges Apple Calendar via macOS EventKit and displays a beautiful Bubble Tea grid.

Key highlights:
- Local Apple Calendar sync
- Monthly/weekly TUI view
- Calculate free slots instantly via `calctl free`
- Exposed as an MCP server with 7 tools
