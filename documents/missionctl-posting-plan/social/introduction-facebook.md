🚀 Announcing missionctl: Giving Local AI Agents 'Hands' via Go CLI/TUI Tools!

AI assistants are great at thinking, but helpless at doing. Spawning browser extensions, cloud synchronizers, and monthly subscription SaaS tools just to connect AI to your daily life shouldn't be the norm. 

missionctl is a local-first, privacy-respecting suite of six Go-based terminal tools. Each tool manages one domain locally in SQLite and exposes a Model Context Protocol (MCP) server so local AI agents (like Claude Desktop) can read and write your data directly.

📦 The 6 Core Tools:
1️⃣ mailctl 📧 — Read, draft, and send email via Apple Mail.
2️⃣ calctl 📅 — List events and find free time slots in Apple Calendar.
3️⃣ taskctl 📝 — Local task manager synchronized with Apple Reminders.
4️⃣ notectl 📓 — Search, read, and write directly into Obsidian vaults.
5️⃣ budgetctl 💰 — Import bank CSVs and tag recurring transactions.
6️⃣ postctl 🐦 — Markdown campaign scheduler for Twitter, LinkedIn, Mastodon, Bluesky, Threads, and Facebook.

🔒 100% Local & Secure:
All credentials, database caches, and logs stay on your local disk. Compile to single, static Go binaries with no extra dependencies.
