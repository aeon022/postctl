📅 Introducing calctl: Calendar Management & Asynchronous Scheduling for Terminal Users

Switching tabs to web calendars is a constant productivity leak. calctl brings calendar scheduling directly into your terminal workspace. Built in Go, it bridges Apple Calendar and exposes a secure MCP server.

Key Features:
• Native macOS Integration: Caches calendar databases locally, bridging to Apple Calendar via EventKit.
• Slot Optimization: Calculate free blocks inside defined working hours instantly.
• YAML Import: Create events from Markdown files with simple headers.
• MCP Server: Exposes 7 tools to Claude to scan, add, or delete events locally.

👉 Check it out on GitHub: https://github.com/aeon022/calctl
