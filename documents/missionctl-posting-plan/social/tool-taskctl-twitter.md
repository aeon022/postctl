Task management from the terminal, synced with Apple Reminders. Meet `taskctl` 📝

A static Go binary replacing python prototypes to give you speed and robust background sync. 🧵
---
Why `taskctl`?
- Bidirectional sync with Apple Reminders (macOS EventKit)
- Background daemon to keep list fresh
- Manage tasks in a beautiful monospace TUI list
- Standardized CLI (`taskctl add`, `taskctl done`)
- Exposes 7 tools to your AI agent via MCP
---
Wired to Claude Desktop, you can ask:
"List all tasks due today, reschedule the overdue ones to tomorrow, and create a new task to check server logs."

AI runs the CLI commands and updates Reminders.
