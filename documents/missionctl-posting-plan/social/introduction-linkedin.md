🚀 Announcing missionctl: Giving Local AI Agents 'Hands' via Monospace CLI/TUI Tools!

AI assistants are great at thinking, but helpless at doing. Spawning browser extensions, cloud synchronizers, and monthly subscription SaaS tools just to connect AI to your daily life shouldn't be the norm. 

missionctl is a local-first, privacy-respecting suite of six Go-based terminal tools built for macOS. Each tool manages one domain locally in SQLite and exposes a Model Context Protocol (MCP) server so local AI agents (like Claude Desktop) can read and write your data directly.

📦 The 6 Core Tools:
1️⃣ mailctl 📧 — Read inbox, search threads, draft and send email via Apple Mail.
2️⃣ calctl 📅 — List events, schedule meetings, and calculate free time slots in Apple Calendar.
3️⃣ taskctl 📝 — Local task manager that synchronizes with macOS Apple Reminders.
4️⃣ notectl 📓 — Search, read, and write directly into your Obsidian/Markdown vaults.
5️⃣ budgetctl 💰 — Import bank CSV exports, categorize, tag, and detect recurring subscriptions.
6️⃣ postctl 🐦 — Markdown-first campaign scheduler for Twitter/X, LinkedIn, Threads, Mastodon, Bluesky, and Facebook.

🔒 100% Local & Secure:
All credentials, database caches, and logs stay on your local disk. Nothing is sent to cloud servers unless you explicitly send an email (mailctl) or publish a post (postctl). 

Every tool compiles to a single, static Go binary with no dependencies (no Docker, no Node, no Python setup).
