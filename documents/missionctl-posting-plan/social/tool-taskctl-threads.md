A static Go binary replacing slow python scripts for seamless background synchronization.

Key highlights:
- Bidirectional Reminders sync (macOS EventKit)
- Background daemon to keep lists fresh
- Monospace Bubble Tea TUI
- Exposes 7 tools to your AI agent via MCP
