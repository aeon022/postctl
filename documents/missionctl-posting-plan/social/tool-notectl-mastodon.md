📓 Meet notectl: Local Markdown & Obsidian note management from the terminal.

Keeping notes in proprietary clouds is a security and lock-in risk. notectl indexes your Markdown files into a local SQLite database for blazing-fast full-text searches.

Features:
• Fast local indexing (`notectl search`)
• Open/append daily notes with a single key
• Vim/Neovim TUI integration
• 7 MCP tools for Claude

👉 https://github.com/aeon022/notectl
