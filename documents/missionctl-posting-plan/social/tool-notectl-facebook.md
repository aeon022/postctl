📓 Introducing notectl: Local Markdown & Obsidian Note Management from the Terminal

Keeping notes in proprietary cloud systems is a security and vendor lock-in risk. notectl lets you index, search, and edit your local Markdown vaults (like Obsidian) directly from the command line.

Key Features:
• Monospace Indexing: Indexes your vault's Markdown files into a local SQLite database for blazing-fast full-text searches.
• Daily Notes: Open or append to today's daily log with a single command.
• Editor Integration: Suspend TUI to edit notes in Vim/Neovim.
• MCP Server: Exposes 7 tools to Claude to read, write, and update notes locally.
