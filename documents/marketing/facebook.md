🚀 Project Launch: postctl — Social Media Management from your Terminal!

Hi everyone! I just released postctl, an open-source tool designed to manage and schedule social media posts directly from local Markdown files. If you love working in the terminal and hate bloated web dashboards, this is for you!

Key Features:
- Markdown-first workflow (YAML frontmatter for platform/scheduling).
- A beautiful keyboard-driven terminal dashboard (TUI) built in Go.
- Seamless external editor support (Vim/Neovim) with live character counters.
- Local SQLite database with encrypted AES-256 backup & multi-device sync.
- Direct posting to Threads, Mastodon, Bluesky, LinkedIn, and Twitter/X.

It's completely free, lightweight, and open source! 

Check it out, run it locally, or check out the code:
👉 https://github.com/aeon022/postctl
