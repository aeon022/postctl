🚀 Announcing postctl — Social Media Management, built for Developers!

Hate leaving your editor and using bloated SaaS web dashboards just to share updates about your projects?

I built postctl to treat social media schedules just like code:

📝 Markdown-First: Write your posts, threads, and metadata inside standard Markdown files with YAML frontmatter.
💻 TUI Dashboard: Run `./postctl tui` to get a gorgeous keyboard-driven dashboard right inside your terminal to manage your drafts and scheduling queue.
⚙️ Vim/Neovim Integration: Edit posts directly from the TUI in Neovim with custom character rulers and live counts.
🔒 Privacy & Local-First: All tokens, campaign plans, and post history are saved in a local SQLite database, encrypted and synced across devices using AES-256-GCM.
💡 Cross-Platform: Publish seamlessly to Threads, Mastodon, Bluesky, LinkedIn, and Twitter/X (bypassing X's high API fees using secure cookie sessions).

postctl is 100% open source, lightweight, and built in Go.

Check out the repository, run it locally, or star the project:
