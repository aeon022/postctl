🚀 Meet postctl: Social Media Management Built for Developers & AI

Tired of clicking through heavy web interfaces just to schedule a post? postctl is a terminal-native CLI and TUI tool written in Go that treats your postings as code.

Key Features:
• Markdown-first: Write posts in your favorite editor (Vim/Neovim integrated!).
• AI-as-Operator: Designed for AI terminal agents to run the full workflow.
• Bilingual (EN/DE) documentation & landing page.
• Robust Headless Fallback: Automated Chrome fallback if API calls get flagged.

Try it now: https://postctl.sh
