Exposing your API keys to cloud databases is a risk you don't need to take. 🔒

postctl stores all credentials locally in SQLite, encrypted with AES-256-GCM derived from your passphrase. No cloud intermediate servers, no telemetry.

GitHub: https://github.com/aeon022/postctl
