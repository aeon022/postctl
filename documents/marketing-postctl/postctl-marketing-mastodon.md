🚀 Meet postctl: Social Media Management Built for Developers & AI

Tired of clicking through heavy web interfaces? postctl is a terminal-native Go CLI/TUI tool treating your postings as code.

Features:
• Markdown-first: Write posts in Vim/Neovim.
• AI-as-Operator: Designed for local AI agents to run the workflow.
• Robust Fallback: Silent Chrome if API requests are blocked.
• 100% Local & Encrypted.

👉 https://github.com/aeon022/postctl
