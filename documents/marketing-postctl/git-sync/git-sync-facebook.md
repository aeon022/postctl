Social Media as Code: Git Integration and Multi-Device Sync 🔄

Why use a proprietary cloud database to sync your scheduled posts? postctl treats your social media queue like code.

Drafts are written in Markdown. To sync your schedule across devices, postctl lets you export your SQLite database into a single encrypted binary backup (AES-256-GCM) that you can safely commit to your private Git repository.
