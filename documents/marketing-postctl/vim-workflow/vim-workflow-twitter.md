## Tweet 2
Built for terminal enthusiasts:

1. Spawn Neovim directly from TUI fields using `ctrl+v`.
2. Live character rulers in Neovim showing platform limits (e.g. 280 for Twitter, 300 for Bluesky).
3. YAML frontmatter bidirectional sync back to TUI upon saving.
