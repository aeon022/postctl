Still managing social media postings through slow, bloated web dashboards? 

It's time to bring scheduling into your native developer workflow. Meet `postctl` — a terminal-native, Markdown-first scheduler and publisher! 🧵
---
With postctl, a post is just a simple Markdown file with YAML frontmatter.

- Draft posts in your favorite local workspace
- Control target platforms and schedule times in frontmatter
- Version control posts in Git and review via PRs!
---
It comes with a fully keyboard-driven Terminal User Interface (TUI):
- Edit posts using an inline text area
- Check auth statuses of Twitter/X, LinkedIn, Threads, Mastodon, Bluesky & Facebook
- View your calendar and history in one place
