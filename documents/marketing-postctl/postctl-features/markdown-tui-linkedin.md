🚀 Announcing postctl: Social Media Management, Built for Developers!

Hate leaving your editor and using heavy SaaS web dashboards just to share project updates? 

postctl brings your social media schedule into your local terminal workflow. By treating posts as code, you can draft, schedule, and sync everything offline in Markdown and SQLite.

Key features:
📝 Markdown-First: Write your posts, threads, and metadata inside standard Markdown files with YAML frontmatter.
💻 TUI Dashboard: A keyboard-driven Terminal User Interface built with Go/Bubble Tea to manage drafts and scheduling queues.
🔒 Privacy & Local-First: All credentials, scheduling queues, and histories are stored in a local SQLite file on your disk.

Stop using bloated SaaS tools. Take control of your distribution from your command line.
