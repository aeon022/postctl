Terminal Social Media Automation: AI-as-Operator 🤖

Say goodbye to bloated web UIs for scheduling posts! postctl is designed with an "AI-First" CLI layout, allowing AI assistants like Claude, ChatGPT, or Antigravity to act as operator scripts.

The AI agent can scrape documentation or websites, generate platform-specific Markdown posts, validate them using `postctl import --dry-run`, and write them directly to your local SQLite queue.
